# 里程碑1：角色属性分配

## 任务目标
编写一个Java程序，让用户为英雄分配属性点，并计算总属性值。

## 要求
1. 程序依次提示用户输入：
   - 英雄名字（字符串）
   - 力量值（整数）
   - 敏捷值（整数）
   - 智力值（整数）
2. 计算总属性点 = 力量 + 敏捷 + 智力。
3. 输出格式为：`总属性点: 数值`（例如 `总属性点: 24`）。

## 进阶挑战（选做）
如果总属性点超过30，输出提示 `属性点超限，请重新分配`，并让用户重新输入（需要循环，可后续学习）。

## 提交要求
- 代码需包含必要注释，变量命名符合Java规范。
- 确保程序能够编译运行，无语法错误。
- 将代码推送到你的GitHub作业仓库的对应文件夹中。

## 评分标准（满分100）
- 功能实现（40分）：程序能正确接收输入并输出总属性点。
- 代码质量（40分）：命名规范，有适当注释，格式清晰。
- 提交完整性（20分）：成功推送代码，提交信息明确。
